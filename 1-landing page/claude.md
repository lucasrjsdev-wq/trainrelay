# Contexto del Proyecto
El proyecto se llama "train relay". Es sitio que promociona una suite de aplicaciones que relacionan entrenadores con atletas, permitiendo la generación de rutinas de entrenamiento y los resultados de esos entrenamientos. Lo que propone esta pagina web es el uso y adopción de una aplicación  de gestión de información de sesiones de entrenamientos de distintas personas vinculanddose con el strava de cada usuario.  

# Reglas
- la carpeta raiz de este proyecto es "1-landing page"
- Usa componentes funcionales y seguros.
- Estilos con Tailwind CSS.
- Al iniciar el proyecto creas un archivo de memoria llamado memory.mp y allí vas a guardar todo lo relevante que te diga o te pida que guardes.
- Todo el código generado debe quedar guardado dentro de la carpeta "code" de este proyecto. Si no está la carpeta, por favor crearla.
