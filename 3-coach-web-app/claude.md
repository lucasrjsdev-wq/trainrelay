# Contexto del Proyecto
El proyecto se llama "train relay". La aplicacion de entrenador que debemos construir es la que va a usar el entrenador que se suscriba a la aplicación. Debe ser una aplicación segura y robusta que permite a los entrenadores registrarse completando todos sus datos para acceder a las funcionalidades. La idea es que el entrenador pueda ver que atletas se registraron, pueda aceptarlos y generarles rutinas de entrenamientos y hacer el correspondiente seguimiento.
