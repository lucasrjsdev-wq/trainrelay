# Contexto del Proyecto
El proyecto se llama "train relay". La aplicacion de administracion que debemos construir es la que va a usar el dueño de la empresa. Debe ser una aplicación segura y robusta que permite a los clientes (entrenadores y atletas) registrarse para acceder a los beneficios. La idea es que el dueño pueda ver que clientes se registraron y pueda ponerse en contacto con ellos para avanzar.  

# Reglas
- la carpeta raiz de este proyecto es "2-admin web app"
- Usa componentes funcionales y seguros.
- Usa el stack Laravel+PHP+React+mySQL
- Al iniciar el proyecto creas un archivo de memoria llamado memory.md y allí vas a guardar todo lo relevante que te diga o te pida que guardes.
- Todo el código generado debe quedar guardado dentro de la carpeta "code" de este proyecto. Si no está la carpeta, por favor crearla.
- Para construir la aplicacion debes usar la plantilla que te dejo en la carpeta "Plantilla" de este proyecto.

# Detalles del proyecto
