-  El sitio debe ser una aplicación web responsive.
- Aplicación debe soportar dos idiomas (ingles y español) con lo cual debes generar todo lo necesario para dicha funcionalidad.

- Para construir la aplicacióm debes usar la plantilla que dejo en la carpeta "plantilla" de este proyecto.

- Para estructurar las funcionalidades de la aplicación debe usar el archivo "use_cases.md" y respetar lo mejor que puedas las indicaciones del mismo. Así mismo cuando te pida construir una nueva funcionalidad, necesito que primero la escribas como un nuevo caso de uso en este archivo, y luego la construyas.
